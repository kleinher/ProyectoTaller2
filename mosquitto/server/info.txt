En este directorio está la configuración del servidor remoto para MOSQUITTO.

*** Instrucciones de instalación del broker ***

(1) Agregar el repositorio de Mosquitto al apt:

$ sudo add-apt-repository ppa:mosquitto-dev/mosquitto-ppa

(2) Instalar el broker/cliente en el servidor:

$ sudo apt install mosquitto mosquitto-clients

Para los clientes se instala solo mosquitto-clients. En ambos casos se 
instalan automáticamente las librerías.

(3) Para agregar seguridad:

$ sudo mosquitto_passwd -c /etc/mosquitto/passwd moquitto

El usuario "mosquitto" ahora tiene contraseña (determinada por el prompt):

Password: *****
Reenter password: *****

Indicarle la nueva configuración al servicio:

$ sudo nano /etc/mosquitto/mosquitto.conf

Escribir y guardar en el archivo anterior:

listener 15883
password_file /etc/mosquitto/passwd

(4) Reiniciar el servicio:

$ sudo systemctl restart mosquitto

(5) Si falla al reiniciar, revisar la config y los logs. Seguramente hay 
problemas con los permisos. Se soluciona:

$ sudo chown mosquitto:mosquitto /etc/mosquitto/passwd
